# ng-cloudblock-blank


## Quick start

There are two ways to install:

### Use the Command Line

Depend on `@delon/cli`, [How install?](http://ng-alain.com/docs/cli)

```bash
ng new  my-project
```

### Clone the Git Repository

```bash

# change directory
cd my-project

# install npm package
ng add ng-cloudblock-blank

# start the serve
ng serve

```
